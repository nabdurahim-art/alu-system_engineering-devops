# SSH - Secure Shell

This project covers the basics of network authentication using SSH keys, configuring a local SSH client, and managing secure access to remote servers.

## Files Description

*   **`0-use_a_private_key`**: A Bash script that connects to an SSH server using a specific private key (`~/.ssh/school`).
*   **`1-create_ssh_key_pair`**: A Bash script that generates a 4096-bit RSA key pair named `school` with the passphrase `betty`.
*   **`2-ssh_config`**: An SSH client configuration file that automatically enforces key-based authentication and disables password fallbacks.
